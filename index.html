Simple and Powerful Architecture

The system consists of two components:

Server

akeb/monitoring

Control center. Web interface, database, API.

Worker

akeb/monitoring-worker

Lightweight agent. Performs checks and sends results to the Server.

Quick Start

  1. Deploy Server: Run the akeb/monitoring image.
  2. Create Worker: In the web interface, create a worker and copy its key.
  3. Launch Worker: On a remote server, execute: 
    docker run -d \
  -e SERVER_URL=https://your-monitoring-server.com \
  -e WORKER_KEY=your_unique_worker_key_here \
  akeb/monitoring-worker
  4. Create monitor: In the UI, specify URL, check type and select workers.
